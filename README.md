## 🎨 Опис проєкту

**THE MET** — адаптивна лендинг-сторінка, зверстана за PDF-макетом музею.  
Мета — показати:

- pixel-perfect верстку,
- акуратну адаптивність,
- чисту семантичну розмітку,
- структурований SCSS без фреймворків.

Сторінка реалізована на **чистому HTML, SCSS і ванільному JavaScript**, розгорнута на **GitHub Pages**.

## ✨ Основні можливості

- 📐 **Pixel-perfect** відповідність дизайну  
- 📱 **Адаптивний дизайн** для desktop / tablet / mobile  
- 🎞 Плавні **hover-ефекти** та мікроанімації  
- 🧩 **Семантична HTML-структура**  
- 🎨 **SCSS-архітектура** з розбиттям на partial-файли  
- 🖼 Використання **зображень з різних секцій** (events, online)  
- 🌐 **Кросбраузерна сумісність**
